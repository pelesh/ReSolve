Welcome to Re::Solve's documentation!
===================================

**Re::Solve** is a library of GPU-resident sparse linear solver. It aims to provide easy-to-use
and maintain solvers for use in complex applications.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
